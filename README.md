## addr.sx

Displays your public address IP with a taste of 90's.

### Usage

#### From a web brower:
Go to [addr.sx](http://addr.sx), or [ip.addr.sx](http://ip.addr.sx) for a plain text result.

#### From a command line:
Type ```curl -L addr.sx``` or ```wget -qO- addr.sx```
